## 二、重点工作完成情况

### 2.1 入职融入与制度学习

1. **完成入职手续办理**：从 base 地前往项目地完成现场报到，签署劳动合同、保密协议、录用条件说明书签收等全部入职文件。
2. **学习公司规章制度及《员工手册》**：重点掌握考勤制度、保密要求、违纪处分标准及试用期考核规则，明确自身权责边界。
3. **了解行政报销流程**：学习公司行政制度，重点熟悉报销流程及规范要求，确保后续费用报销合规高效。

### 2.2 开发环境搭建与工具准备

围绕 EAP 系统开发所需的接口调试和设备通讯能力，完成以下工具的安装与配置：

1. **Apifox**：HTTP 接口调试与管理，用于对接 MES 系统接口需求。
2. **OPC UA 测试工具**：验证 OPC UA 通讯连接及数据读写。
3. **PLC 协议测试工具**：西门子/欧姆龙等 PLC 通讯调试。
4. **Socket 测试工具**：MODBUS、SECS/GEM 等 Socket 通讯协议的连接与报文验证。

### 2.3 EAP 系统学习与现场实践

1. **参加 EAP 系统软件培训**：学习 EAP 系统的配置与使用方法，建立对系统整体架构的认知。
2. **学习设备访谈方法论**：掌握 EAP 系统对接各设备的技术要点——通讯协议选型策略（WebAPI 优先，OPC UA / Modbus 备选）、数据上报格式、报警前缀同步机制，为后续开发与 MES 对接奠定技术基础。
3. **研读 MES 接口文档**：通过接口文档初步掌握 EAP 系统与 MES 的数据交互逻辑，明确后续开发需关注的接口字段与调用方式。
4. **了解项目进度与交付节点**：结合项目计划了解当前项目所处阶段，便于合理安排个人工作节奏。
5. **厂房 10 现场走访**：实地了解车间设备布局与线体运作模式，建立对设备访谈的实际场景认知，为后续携带 Checklist 针对性提问做好准备。
6. **学习 PCB 产线自动化全流程**：涵盖上板→传输→加工→检测→下板完整链路，掌握上下板机自动化流程及投收板机读码校验规则（PNL 码→料号/LOT 校验→工单匹配）。
7. **梳理 AGV 安全信号流转逻辑**：明确 EAP → MES → 设备三方间的信号流转与互锁机制。
8. **学习 Kyber Admin 工业软件**：掌握设备连接管理、参数配置及日志查看等核心功能。
9. **熟悉团队 Apifox API 接口库**：了解 EAP 与 MES、设备间的数据联通方式，为后续接口开发建立全局视角。
